# 国际化 (I18n)

`@ant-design/agentic-ui` 提供了完整的国际化解决方案，支持中英文切换，自动检测用户语言偏好，并与 Ant Design 的国际化系统无缝集成。

## 核心特性

- 🌍 **多语言支持**：内置中文和英文语言包
- 🔄 **自动检测**：智能检测用户浏览器语言设置
- 💾 **持久化**：自动保存用户语言选择到 localStorage
- 🎨 **集成 Ant Design**：与 Ant Design ConfigProvider 同步
- 🎯 **类型安全**：完整的 TypeScript 类型定义
- 🔌 **易于扩展**：支持自定义语言包

## 快速开始

### 基础使用

<code src="../demos/i18n-basic-demo.tsx"></code>

```tsx | pure
import { I18nProvide, useLanguage } from '@ant-design/agentic-ui';

// 1. 在应用根组件中使用 I18nProvide
function App() {
  return (
    <I18nProvide>
      <YourComponents />
    </I18nProvide>
  );
}

// 2. 在子组件中使用 useLanguage Hook
function YourComponent() {
  const { locale, language, setLanguage, toggleLanguage } = useLanguage();

  return (
    <div>
      <p>{locale.welcome}</p>
      <button onClick={toggleLanguage}>
        {language === 'zh-CN' ? 'Switch to English' : '切换到中文'}
      </button>
    </div>
  );
}
```

## API 文档

### I18nProvide

国际化提供者组件，为整个应用提供国际化功能。

#### Props

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| children | 子组件 | `ReactNode` | - | - |
| locale | 自定义语言包 | `typeof cnLabels` | - | - |
| defaultLanguage | 默认语言 | `'zh-CN' \| 'en-US'` | `'zh-CN'` | - |
| autoDetect | 是否自动检测语言 | `boolean` | `true` | - |

#### 示例

```tsx | pure
import { I18nProvide } from '@ant-design/agentic-ui';

// 基本使用
<I18nProvide>
  <App />
</I18nProvide>

// 指定默认语言
<I18nProvide defaultLanguage="en-US">
  <App />
</I18nProvide>

// 关闭自动检测
<I18nProvide autoDetect={false} defaultLanguage="zh-CN">
  <App />
</I18nProvide>
```

### useLanguage Hook

语言切换 Hook，提供语言状态和切换方法。

#### 返回值

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| language | 当前语言代码 | `'zh-CN' \| 'en-US'` | - | - |
| locale | 当前语言的标签对象 | `LocalKeys` | - | - |
| setLanguage | 设置语言的方法 | `(language: 'zh-CN' \| 'en-US') => void` | - | - |
| toggleLanguage | 切换中英文的方法 | `() => void` | - | - |
| isChinese | 是否为中文 | `boolean` | - | - |
| isEnglish | 是否为英文 | `boolean` | - | - |

#### 示例

```tsx | pure
import { useLanguage } from '@ant-design/agentic-ui';

function LanguageSwitcher() {
  const {
    language,
    locale,
    setLanguage,
    toggleLanguage,
    isChinese,
    isEnglish,
  } = useLanguage();

  return (
    <div>
      {/* 显示当前语言 */}
      <p>当前语言: {language}</p>

      {/* 使用语言包中的文本 */}
      <h1>{locale.welcome}</h1>
      <p>{locale.description}</p>

      {/* 切换语言 */}
      <button onClick={toggleLanguage}>{locale.switchLanguage}</button>

      {/* 设置特定语言 */}
      <button onClick={() => setLanguage('en-US')}>English</button>
      <button onClick={() => setLanguage('zh-CN')}>中文</button>

      {/* 条件渲染 */}
      {isChinese && <div>这是中文内容</div>}
      {isEnglish && <div>This is English content</div>}
    </div>
  );
}
```

### 工具函数

#### detectUserLanguage

自动检测用户语言偏好。

```tsx | pure
import { detectUserLanguage } from '@ant-design/agentic-ui';

const language = detectUserLanguage();
console.log(language); // 'zh-CN' 或 'en-US'
```

检测优先级：

1. localStorage 中保存的用户选择
2. Ant Design ConfigProvider 的 locale
3. 浏览器语言设置
4. 默认语言（中文）

#### getLocaleByLanguage

根据语言代码获取对应的语言包。

```tsx | pure
import { getLocaleByLanguage } from '@ant-design/agentic-ui';

const locale = getLocaleByLanguage('zh-CN');
console.log(locale.welcome); // '欢迎'
```

#### saveUserLanguage

保存用户语言选择到 localStorage。

```tsx | pure
import { saveUserLanguage } from '@ant-design/agentic-ui';

saveUserLanguage('en-US');
```

#### compileTemplate

编译模板字符串，支持变量替换。

<code src="../demos/i18n-template-demo.tsx"></code>

```tsx | pure
import { compileTemplate } from '@ant-design/agentic-ui';

// 基本使用
const message = compileTemplate('你好，${name}！', { name: '世界' });
console.log(message); // '你好，世界！'

// 缺少变量时显示占位符
const message2 = compileTemplate('你好，${name}！');
console.log(message2); // '你好，[name]！'

// 实际应用场景
const { locale } = useLanguage();
const taskMessage = compileTemplate(locale.inProgressTask, {
  taskName: '数据分析',
});
console.log(taskMessage); // '正在进行数据分析任务'
```

### 自定义语言包

你可以扩展或覆盖默认语言包：

```tsx | pure
import { I18nProvide, cnLabels } from '@ant-design/agentic-ui';

// 扩展语言包
const customLocale = {
  ...cnLabels,
  customKey: '自定义文本',
  anotherKey: '另一个文本',
};

function App() {
  return (
    <I18nProvide locale={customLocale}>
      <YourComponents />
    </I18nProvide>
  );
}
```

## 与 Ant Design 集成

`I18nProvide` 会自动与 Ant Design 的 `ConfigProvider` 同步语言设置。

```tsx | pure
import { ConfigProvider } from 'antd';
import zhCN from 'antd/locale/zh_CN';
import enUS from 'antd/locale/en_US';
import { I18nProvide, useLanguage } from '@ant-design/agentic-ui';

function App() {
  const { language } = useLanguage();
  const antdLocale = language === 'zh-CN' ? zhCN : enUS;

  return (
    <ConfigProvider locale={antdLocale}>
      <I18nProvide>
        <YourComponents />
      </I18nProvide>
    </ConfigProvider>
  );
}
```

### 完整的集成示例

```tsx | pure
import { ConfigProvider, Button } from 'antd';
import zhCN from 'antd/locale/zh_CN';
import enUS from 'antd/locale/en_US';
import { I18nProvide, useLanguage } from '@ant-design/agentic-ui';

function AppContent() {
  const { language, locale, toggleLanguage } = useLanguage();

  return (
    <div>
      <h1>{locale.welcome}</h1>
      <Button onClick={toggleLanguage}>{locale.switchLanguage}</Button>
    </div>
  );
}

function App() {
  return (
    <I18nProvide>
      <ConfigProviderWrapper>
        <AppContent />
      </ConfigProviderWrapper>
    </I18nProvide>
  );
}

function ConfigProviderWrapper({ children }) {
  const { language } = useLanguage();
  const antdLocale = language === 'zh-CN' ? zhCN : enUS;

  return <ConfigProvider locale={antdLocale}>{children}</ConfigProvider>;
}
```

### 3. 动态消息提示

```tsx | pure
import { message } from 'antd';
import { useLanguage, compileTemplate } from '@ant-design/agentic-ui';

function useNotification() {
  const { locale } = useLanguage();

  const showSuccess = (name: string) => {
    const msg = compileTemplate(locale.saveSuccess, { name });
    message.success(msg);
  };

  const showError = (errorMsg: string) => {
    const msg = compileTemplate(locale.errorOccurred, { error: errorMsg });
    message.error(msg);
  };

  return { showSuccess, showError };
}

// 使用
function MyComponent() {
  const { showSuccess, showError } = useNotification();

  const handleSave = async () => {
    try {
      // 保存逻辑
      showSuccess('用户数据');
    } catch (error) {
      showError(error.message);
    }
  };

  return <Button onClick={handleSave}>保存</Button>;
}
```

## 最佳实践

### 1. 统一语言管理

创建一个统一的语言管理 Hook：

```tsx | pure
import { useLanguage, compileTemplate } from '@ant-design/agentic-ui';

export function useI18n() {
  const { locale, language, setLanguage, toggleLanguage } = useLanguage();

  // 包装 compileTemplate 方法
  const t = (key: keyof typeof locale, variables?: Record<string, string>) => {
    const template = locale[key];
    return variables ? compileTemplate(template, variables) : template;
  };

  return {
    t,
    locale,
    language,
    setLanguage,
    toggleLanguage,
  };
}

// 使用
function MyComponent() {
  const { t } = useI18n();

  return (
    <div>
      <h1>{t('welcome')}</h1>
      <p>{t('inProgressTask', { taskName: '数据分析' })}</p>
    </div>
  );
}
```

### 2. 类型安全的语言包

```tsx | pure

// 扩展语言包类型
interface CustomLocale extends LocalKeys {
  customKey: string;
  anotherKey: string;
}

const customCnLabels: CustomLocale = {
  ...cnLabels,
  customKey: '自定义文本',
  anotherKey: '另一个文本',
};
```

### 3. 懒加载语言包

对于大型应用，可以懒加载语言包：

```tsx | pure
import { useState, useEffect } from 'react';
import { I18nProvide } from '@ant-design/agentic-ui';

function App() {
  const [locale, setLocale] = useState(null);

  useEffect(() => {
    // 动态加载语言包
    import('./locales/custom-locale').then((module) => {
      setLocale(module.default);
    });
  }, []);

  if (!locale) {
    return <div>Loading...</div>;
  }

  return (
    <I18nProvide locale={locale}>
      <YourComponents />
    </I18nProvide>
  );
}
```

### 5. 性能优化

使用 `useMemo` 优化语言相关的计算：

```tsx | pure
import { useMemo } from 'react';
import { useLanguage } from '@ant-design/agentic-ui';

function DataTable() {
  const { locale } = useLanguage();

  // 缓存列配置，避免重复计算
  const columns = useMemo(
    () => [
      { title: locale.name, dataIndex: 'name' },
      { title: locale.age, dataIndex: 'age' },
      { title: locale.address, dataIndex: 'address' },
    ],
    [locale],
  );

  return <Table columns={columns} />;
}
```

## 常见问题

### 1. 如何添加新的语言？

目前仅支持中英文，如需添加其他语言：

```tsx | pure
// 创建新的语言包
const jpLabels = {
  welcome: 'ようこそ',
  // ... 其他标签
};

// 扩展类型定义
type Language = 'zh-CN' | 'en-US' | 'ja-JP';
```

### 2. 如何持久化语言选择？

语言选择会自动保存到 localStorage，键名为 `agentic-ui-language`。

```tsx | pure
// 手动读取
const savedLanguage = localStorage.getItem('agentic-ui-language');

// 手动设置
localStorage.setItem('agentic-ui-language', 'en-US');
```

### 3. 如何覆盖默认语言检测？

设置 `autoDetect={false}` 并指定 `defaultLanguage`：

```tsx | pure
<I18nProvide autoDetect={false} defaultLanguage="en-US">
  <App />
</I18nProvide>
```

### 4. useLanguage 必须在 I18nProvide 内使用吗？

是的，`useLanguage` 必须在 `I18nProvide` 组件内部使用，否则会抛出错误。

```tsx | pure
// ❌ 错误用法
function App() {
  const { locale } = useLanguage(); // 会报错
  return <div>{locale.welcome}</div>;
}

// ✅ 正确用法
function App() {
  return (
    <I18nProvide>
      <Content />
    </I18nProvide>
  );
}

function Content() {
  const { locale } = useLanguage(); // 正确
  return <div>{locale.welcome}</div>;
}
```

## 注意事项

1. **Context 嵌套**：确保 `I18nProvide` 在组件树的顶层
2. **性能考虑**：语言切换会触发组件重渲染，注意使用 `useMemo` 优化
3. **SSR 支持**：服务端渲染时注意 localStorage 的访问
4. **类型安全**：使用 TypeScript 确保语言包的类型安全
5. **测试环境**：测试环境默认使用英文，可通过 `autoDetect={false}` 修改
